# NAICS Direct

Federal government bids filtered by industry — built for contractors, distributors, and resellers who are tired of digging through SAM.gov manually.

## Features
- 🔍 Live SAM.gov bid feed filtered into 8 industry niches
- ⏰ Urgency-coded deadlines (red ≤2d, amber ≤7d, green >7d)
- 📊 Historical pricing context
- 💡 Built for small businesses and SDVOSB contractors

## Stack
- **Next.js 15** (App Router, Turbopack)
- **TypeScript** + **Tailwind CSS** + **shadcn/ui**
- **Stripe** for subscriptions (3 tiers: $14/$29/$49/mo beta)
